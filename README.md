# 🚀 Confio Frontend 🚀

Hey there! 👋 Welcome to the Confio Frontend repo! This is helping for ur starting project, making sure all the UI stuff runs smooth and easy. We're using some awesome tech here to keep everything looking fresh and blazing fast! ✨

## 🛠️ What's Inside?

- **Next.js & React 19** — The best tools for building super fast websites.
- **TypeScript** — For that sweet, sweet type safety.
- **Tailwind CSS & Shadcn/ui** — Making things pretty without the CSS headaches.
- **TanStack Query & Axios** — Fetching data like an absolute boss.
- **Formik & Yup** — Handling forms and validation the easy way.

## 🚀 How to Get Started

Follow these simple steps and your frontend will be alive:

### 1. Install Dependencies 📦
First things first, grab all the packages:
" " "bash
npm install
" " "

### 2. Setup Environment Variables 🌍
Make a .env file for your API URLs (like NEXT_PUBLIC_API_URL).

### 3. Run the Development Server! 🎉
Let's see this UI in action:
" " "bash
npm run dev
" " "
Now, open [http://localhost:3000](http://localhost:3000) in your browser. 🎈

## 🚢 Building for Production
When it's time to ship, just build and start:
" " "bash
npm run build
npm start
" " "

Have fun coding UI! 💅✨
